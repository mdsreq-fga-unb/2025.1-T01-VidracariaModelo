# 2025.1-T01-VidracariaModelo
Repositório de projeto da disciplina de REQ-T1.


## Comandos para rodar o front do projeto

Para rodar o projeto é preciso ter o gerenciador de pacotes **npm** instalado na máquina. Após instalar o gerenciador, utilize os comandos:

Navegue até script/frontend e execute os comandos:

```
$ npm install
```

```
$ npm run dev
```
