## Regras de Negócio (RN)
| ID | Regra de Negócio |
|---|---|
| RN01 | Os agendamentos listados tem que ter sido feitos no máximo com uma semana. |
| RN02 | Agendamentos só podem ser feitos em horários disponíveis e dentro do expediente da empresa. |
| RN03 | Clientes podem cancelar agendamentos com no mínimo 12h de antecedência. |
| RN04 | Agendamentos cancelados geram notificações para o cliente e o administrador. |
| RN05 | Apenas administradores podem criar ou consultar agendamentos de outros usuários. |
| RN06 | Somente administradores podem editar agendamentos enviados ao cliente. |
| RN07 | O sistema deve alertar automaticamente sobre agendamentos não confirmados nos próximos 3 dias. |
| RN08 | Os orçamentos devem expirar em no máximo 15 dias. |
| RN09 | Orçamentos devem estar associados a um cliente e podem ser filtrados por cliente, status, etc. |
| RN10 | Vendas e recibos devem estar relacionados a um cliente e orçamento. |
| RN11 | Apenas administradores podem criar, editar ou excluir recibos já registrados. |
| RN12 | Vendas e recibos podem ser exportados em PDF com os dados registrados. |
| RN13 | Apenas administradores podem registrar, editar ou excluir pagamentos. |
| RN14 | Toda despesa registrada deve ser categorizada para compor relatórios financeiros. |
| RN15 | Todo serviço (venda e atendimento) deve ser associado ao cliente. |
| RN16 | Somente administradores podem incluir ou editar anotações internas no perfil do cliente. |
| RN17 | Somente administradores podem criar e/ou consultar gráficos de desempenho. |
| RN18 | Clientes podem avaliar serviços após sua conclusão. |
| RN19 | Os agendamentos podem ser cancelados por administradores e pelos clientes. |
| RN20 | Apenas os administradores podem criar, editar ou excluir perfis de cliente. |
